**547.朋友圈**

思路：图的深度优先遍历，求连通分量


```java
class Solution {

    private int[][] M;
    boolean[] visited;

    public int findCircleNum(int[][] M) {
        this.M = M;
        visited = new boolean[M.length];
        int ccount = 0;
        for(int i = 0; i < M.length; i++){
            if(!visited[i]){
                dfs(i);
                ccount++;
            }
        }
        return ccount;
    }

    private void dfs(int n){
        visited[n] = true;
        for(int i = 0; i < M[n].length; i++){
            if(M[n][i] == 1 && !visited[i]){
                dfs(i);
            }
        }
    }

}
```
